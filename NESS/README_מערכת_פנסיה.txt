# מדריך שימוש במערכת לחילוץ נתוני פנסיה

## 🚀 הפעלה פשוטה

1. הכנס את קבצי המסלקה (XML) לתיקייה הנוכחית או לתיקיית DATA
2. לחץ כפול על הקובץ `run_pension_system.bat`
3. המערכת תחפש אוטומטית את כל הקבצים ותחלץ את הנתונים

## 📋 מה המערכת עושה

✅ מחפשת את כל קבצי הXML בתיקיות השונות
✅ מחלצת את כל תכניות הפנסיה מכל הקבצים
✅ יוצרת דוח אקסל מסודר עם כל הנתונים
✅ מציגה סיכום מפורט במסך

## 📁 קבצים שחשוב לדעת

- `universal_pension_system_v2.py` - הקובץ הראשי של המערכת
- `run_pension_system.bat` - קובץ להפעלה קלה
- `universal_pension_report.xlsx` - הדוח שנוצר

## 🎯 תוצאות

המערכת תציג:
- מספר תכניות הפנסיה שנמצאו
- הסכום הכולל של כל היתרות
- טבלת אקסל מפורטת עם כל הנתונים

## 💡 טיפים

- המערכת עובדת עם כל סוגי קבצי המסלקה
- היא מוצאת אוטומטית את כל היתרות הרלוונטיות
- הדוח נוצר באופן אוטומטי בתיקייה

**המערכת מוכנה לשימוש! 🎉**
